# Screenshot Worklog Tier Plans

## Free

- Local screenshot time-block recognition stub.
- JSON/file-based storage model and public configuration examples.
- Basic statistics and manual review workflow.
- No cloud sync, billing automation, or customer data integrations.

## Plus

- Private OCR/AI-assisted recognition pipeline.
- Better search, project tagging, recurring work patterns, and richer reports.
- Optional local database schema and batch import/export flows.
- Private implementation lives in `screenshot-worklog-plus-pro`.

## Pro

- Approval workflow for billable records.
- Anomaly detection, client/team reporting, audit trails, and policy checks.
- Human review before finalizing billable or externally shared records.
- Private governance implementation lives in `screenshot-worklog-plus-pro`.

## Repository Split

- `screenshot-worklog`: public Free tier only.
- `screenshot-worklog-plus-pro`: private Plus and Pro development.
